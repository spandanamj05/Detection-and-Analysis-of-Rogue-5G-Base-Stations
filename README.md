# Rogue 5G Base Station Detection System

## Overview
This project focuses on detecting and analyzing rogue/fake 5G base stations for cybersecurity and telecom security research purposes.

The system monitors abnormal network behavior and generates alerts for suspicious base station activity.

---

## Technologies Used
- Linux
- MongoDB
- Virtual Machines
- Networking tools
- Python/Scripting
- 5G simulation environment

---

## Features
- Subscriber monitoring
- Base station behavior analysis
- Alert generation
- Network activity logging
- Security-focused simulation

## Environment Setup
- Ubuntu VM
- MongoDB
- Python
- VirtualBox
- Simulated 5G network environment

## Objective
To understand telecom security vulnerabilities and develop methods to detect suspicious network behavior in 5G environments.
